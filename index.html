How We Detect Without Tracking

We run on Cloudflare edge. Your IP and geolocation are processed in memory and never stored. No tracking cookies. Primary source is our edge API; if it is unavailable, we fall back to Cloudflare /cdn-cgi/trace, then ipify as a last resort.

IPv4 vs IPv6 — Simple

  • IPv4: 32-bit addresses (e.g., 203.0.113.7).
  • IPv6: 128-bit addresses (e.g., 2001:db8::1).
  • Seeing both is normal — some networks prefer one over the other.
